Detailed Description

Function for escaping and unescaping string using back-slash (\). Implemented escape characters:

Escaped character Description Byte in ASCII encoding
\' single quote 0x27
\" <td> double quote <td align="right"> 0x22 <tr><td> \\? <td> question mark <td align="right"> 0x3f <tr><td> \\\\ <td> backslash <td align="right"> 0x5c <tr><td> \\a <td> audible bell <td align="right"> 0x07 <tr><td> \\b <td> backspace <td align="right"> 0x08 <tr><td> \\f <td> form feed - new page <td align="right"> 0x0c <tr><td> \\n <td> line feed - new line <td align="right"> 0x0a <tr><td> \\r <td> carriage return <td align="right"> 0x0d <tr><td> \\t <td> horizontal tab <td align="right"> 0x09 <tr><td> \\v <td> vertical tab <td align="right"> 0x0b
